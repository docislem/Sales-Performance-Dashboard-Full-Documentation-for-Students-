# Sales Performance Dashboard 
Level: Beginner <br>
Tool: Microsoft Excel <br>
Duration: 1–2 days <br>
Goal: Learn how to clean data, analyze it using pivot tables, and create a professional dashboard.
## Data Cleaning
### Check for missing values
If the filter is not activated for the header row:<br>
1-Select the header row.<br>
2-Go to the Home tab.<br>
3-Click Sort & Filter.<br>
4-Choose Filter to enable filtering<br>
